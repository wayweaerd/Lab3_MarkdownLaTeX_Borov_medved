#### Расширенный Markdown: Дополнительные возможности

#### Чекбоксы   

#### Мини-план на день

-[x] Атжумания делать
-[] Пресс качат
-[] Бегат

#### Сноска

Markdown полезен в разработке[^markdown]

#### Alert-блок

> [!TIP] 
полезный совет
> [!IMPORTANT] 
важная информация

#### Escape-символы

* Заголовок без форматирования: `\# это не заголовок`
* Звездочка как текст: `\*Просто звездочка\*`

#### HTML в Markdown

```HTML
<p style="background-color: #f8f9fa; padding: 15px; border-radius: 5px;">
    Это параграф оформлен с помощью <strong>HTML-тега</strong> внитри Markdown-документа.
</p>
```
