___

Markdown (маркдаун) — лёгкий язык разметки для текстовых документов. Создан Джоном Грубером и Аароном Шварцем в 2004 году. 
___
Некоторые возможности Markdown:
создание заголовков; 
форматирование текста (выделение курсивом или жирным) с помощью звёздочек и нижнего подчёркивания; 
создание списков (нумерованных и маркированных); 
вставка элементов (горизонтальные линии, ссылки, изображения). 
Ключевая особенность Markdown — сохранение читаемости документа даже в исходном виде, без преобразования в конечный формат. 
___
Пример формирования: заголовки
