## Лабораторная работа №3. Работа с Markdown-разметкой и базовое использование LaTeX в документации проекта

#### Markdown (маркдаун) — лёгкий язык разметки для текстовых документов. Создан Джоном Грубером и Аароном Шварцем в 2004 году. 

#### Markdown применяется в разных сферах, например: 
Программирование
Написание и форматирование текстов
Подготовка учебных материалов
Ведение задач и заметок
Работа с текстом в мессенджерах
Системы управления контентом (CMS)
GitHub, Stack Overflow и другие ресурсы.

#### Markdown нужен разработчикам для форматирования текста в разных областях: 
документации проектов, комментариев в системах контроля версий, заметок и публикаций в блогах. Язык разметки позволяет структурировать текст, добавлять заголовки, списки и блоки кода, не используя сложные текстовые процессоры.
